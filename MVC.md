# What

## MVC 패턴이란?
소프트웨어를 Model, View, Controller라는 3가지 구성 요소로 구분한 개발 방법론.

사용자 인터페이스, 데이터 및 논리 제어를 구현하는데 널리 사용되는 소프트웨어 [[디자인 패턴]]이다. 소프트웨어의 비즈니스 로직과 화면을 구분하는데 중점을 두고 있으며, 이러한 [[관심사의 분리]] 는 더나은 업무의 분리와 향상된 관리를 제공한다.
